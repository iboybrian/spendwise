# Project Context: SpendWise
## Tech Stack & Architecture

Frontend: Expo / React Native (TypeScript, Expo Router)

Backend/Database: Supabase (PostgreSQL, Edge Functions via Deno)

State Management: Zustand

Styling: Native React Native StyleSheet/Flexbox (Light/Dark mode, Deep Indigo primary)

## Core Feature Implementation Rules

Authentication: Supabase Auth (Email/Password & Google OAuth).

AI Integration: Groq API (Llama 3) via Supabase Edge Functions.

## Coding Standards

Brevity: NEVER output full boilerplate code. Output ONLY the exact lines of code that need changing. Do not explain the code unless asked.

Error Handling: Always use try/catch and React Native Alert.alert() for user feedback.

No Hallucinations: If a package isn't in package.json, ask before installing.

## Restrictions

Do not delete files without explicit permission.

Do not modify .env or app.json autonomously.
